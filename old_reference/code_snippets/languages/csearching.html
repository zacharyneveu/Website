Searching Algorithms

Binary Search

Binary search searches through a sorted array, vector etc. with a runtime of O(log(n)). 
				
template<class T>
int binarySearch(T A[], int first, int last, T target)
{
	if(first > last)
		return -1;
	else
	{
		int mid=(first+last)/2;
		if(target == A[mid])
			return mid;
		else if(target < A[mid])
			return binarySearch(A, first, mid-1, target);
		else return binarySearch(A, mid+1, last, target);
	}
}

Linear Search

Linear search is a simple method to determine if an element with a specified value exists in an unsorted array, vector, or other data structure. This algorithm has a runtime of O(n), however, so for sorted data, binary search should be used. 
					
template<class T>
int linearSearch(T A[], T target)
{
	for (int i=0; i<(sizeof(A)/sizeof(T)); i++)
	{
		if (A[i]==target)
			return i;
		else
			continue
	}
	return -1; //return -1 if not found
	cout<<"Target not found"<<endl;
}
