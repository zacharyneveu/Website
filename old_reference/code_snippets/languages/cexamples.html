Recursive Factorial

This is a simple code snippet to calculate the factorial of a number using recursion. 
				
int fact(int x) {
	if(x == 1 || x == 0)
		return 1;
	else
		return x * fact(x-1);
}

Recursive Fibonacci

This is a simple function to calculate the factorial of a number. This function is a good example of how recursive algorithms can look very simple and elegant, but it is also an example of how they can be inefficient, recalculating information many times over, and becoming unusably slow for even relatively small inputs (As low as about 50 on my machine). 
					
int fib(int x) {
	if(x == 1 || x == 0)
		return 1;
	else
		return fib(x-1) + fib(x-2);
}
